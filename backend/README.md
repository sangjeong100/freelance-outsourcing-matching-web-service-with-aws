## init

```
$ npm i nodemon  (backend 코드 수정시, 자동으로 서버 재시작)
$ npm install --save express (express 모듈 사용)
$ npm i express-session
$ npm i cors
$ npm i morgan
$ npm install cookie-parser
$ npm i aws-sdk
$ npm i mongoose
$ npm i mongoose-auto-increment
$ npm i http-errors
$ npm i express-fileupload
$ npm i pm2 -g

```

## 실행
```
$npm start
```
